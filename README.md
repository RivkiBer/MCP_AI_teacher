# 🎓 עוזר למידה אישי – AI Learning Assistant

אפליקציית **Streamlit** חכמה ליצירת שיעורים מותאמים אישית באמצעות Gemini AI.  
המערכת מייצרת:

- 📖 שיעור קצר ומתומצת לפי נושא ורמת קושי  
- 📝 בוחן אמריקאי אוטומטי (3 שאלות)  
- 🏆 חישוב ציון ומשוב מיידי  
- 💾 שמירת סיכום מקומי כקובץ טקסט  

---

## 🚀 פיצ'רים עיקריים

### ✅ יצירת שיעור מותאם אישית
המשתמש בוחר:
- נושא לימוד
- רמת קושי:
  - ילדים
  - נוער
  - מבוגרים

המערכת שולחת בקשה ל-Gemini ומחזירה:
- תוכן לימודי מותאם לרמה
- 3 שאלות אמריקאיות בפורמט מובנה

---

### ✅ בוחן אינטראקטיבי
- בחירת תשובות דרך Radio Buttons
- חישוב ציון באחוזים
- משוב מיידי לכל שאלה
- אנימציית בלונים לציון 100 🎈
